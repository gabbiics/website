                <h2>C# Conditions and if else statements</h2>
                <p>A conditional statement that executes a block of code when a condition returns a Boolean value of true.</p>
                <p>Can have conditions of any expression, so long as it returns Boolean values (true or false).</p>
                <p>C# supports the usual logical conditions from mathematics:</p>
                <ul>
                    <li>Less than: a < b</li>
                    <li>Less than or equal to: a <= b</li>
                    <li>Greater than: a > b</li>
                    <li>Greater than or equal to: a >= b</li>
                    <li>Equal to a == b</li>
                    <li>Not Equal to: a != b</li>
                </ul>
            </div>

        <div class="basic-concepts">
            <h2>The else Clause</h2>
            <div>
            <p>A clause that can be used for the for the if statement</p>
            <p>Can be specified to execute a block of code when the condition in the if block returns a false value.</p> <br>

            <h2>Nested if-else Statements</h2>
            <p>These are if-else statements within another if-else statement.</p>
            <p>Programmers can nest an infinite amount of if-else statements.</p> <br>

            <h2>if-else Statements</h2>
            <p>A compound conditional statement that can be used to help a program to decide among three (3) or more actions.</p>
            <p>An if statement can have zero (0) or more else-if's clauses and they must come before the last else clause, which is optional.</p> 
            <p>Once an else-if clause succeeds, none of the remaining else-if's or else clauses will be tested.</p><br>

            <h2>While loop</h2>
            <p>Repeatedly executes a block of code as long as a given condition is satisfied</p>
            <h2>For loop</h2>
            <p>A type of loop that executes a statement for a specific number of times.</p>
            <h2>Do-while loop</h2>
            <p>A loop structure that is fundamentally similar to the while loop</p><br>
